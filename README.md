# MERN Estate

A full-stack real estate web application where users can browse properties, create accounts, and promote their own real estate listings. Users can add property details including images, price, number of rooms, and sale/rent status.



## Features
- Browse properties available for rent or sale
- User registration and authentication
- Add and manage personal property listings
- Upload images for properties
- Property search and filtering (by price, rooms, type)
- Responsive design using Tailwind CSS

## Tech Stack
- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB with Mongoose
- **Authentication:** JWT (JSON Web Tokens)
- **File Storage:** Local/Cloud for images
- **Styling:** Tailwind CSS
